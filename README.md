# pandas 課題

このリポジトリは、pandas の基本操作を練習するための課題です。  
`task01` から `task10` まで、1つずつフォルダが分かれています。

各課題フォルダに移動して、Pythonファイルを実行してください。

```bash
cd task01
python task01.py
```

別の課題を実行するときは、いったん親フォルダに戻ってから移動します。

```bash
cd ..
cd task02
python task02.py
```

## インストール

最初に必要なライブラリをインストールします。

```bash
pip install -r requirements.txt
```
